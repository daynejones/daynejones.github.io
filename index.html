About me

I am an ambitious full stack engineer that leans slightly toward the front end. I design and build a wide range of applications, tools, and games for Livingly Media Inc and almost as many for my various side projects. Sometimes that means implementing a new javascript framework and building an API, sometimes coding up a webpage from a mockup, and sometimes its designing and implementing sensical and performant data models.

My favorite languages to work with are Javascript, HTML5, CSS3, SQL, PHP, Python, and Ruby. On top of these languages, I love working with Backbone, Angular, Django, Rails, Git, Unix, MongoDB, Node, Bootstrap and the Bourbon family though I'm always testing out the latest technology in my spare time.

I am opinionated about writing code that is readable, well commented, and full of helpful variable and function names. Performant systems with extensive testing make me incredibly happy. When the browser is involved, cross-platform compatibility is my top priority.

When I'm not writing code for Livingly, I'm writing code or DIYs for Howchoo.com. When I'm not doing that, I'm freelancing. When I'm not doing that, I'm learning the latest new framework or language. When I'm not doing that, I'm probably hiking, reading, making coffee, or with my amazing family.

Projects I'm proud of

Howchoo

Live Site
Tech Used - Python, Django, Backbone, jQuery, the Bourbon family

Howchoo is a DIY network. I'm one of the 2 engineers who built and actively maintain the entire application. At Howchoo, users build and share do it yourself guides covering a wide range of topics from carpentry to programming. This is an ongoing project that gets a significant amount of traffic. One part of the application I'm proud of is the Guide publishing tool. Utilizing Django's formsets on the backend and a custom built Backbone plugin on the front end, I was able to build a dynamic tool that is great for the user and very flexible and easy to use for the developer.
